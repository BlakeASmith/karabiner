
# Karabiner Elements Configuration

## Overview

### Launcher Mode

Simultanious `a` + `l` triggers launcher mode

In this mode each key is set to open a specific app.

### Symbols & Arrows Layer

- Tap the Hyper key (Caps Lock) + `h` to enter a sticky symbols layer.
- Exit with `escape`, `caps_lock`, `return`, or `space`.
- `h`/`j`/`k`/`l` emit the arrow keys.
- Other home-row keys output programming punctuation such as `()[]{}`, `<>`, quotes, math operators (`- _ = +`), `/ ?`, `\ |`, and `` ` ~ ``.


### App Specific Modes

Simultanious `d` + `k` triggers app-specific modes
